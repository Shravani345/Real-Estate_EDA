# Real Estate Exploratory Data Analysis (EDA) Project

- This project performs an in-depth Exploratory Data Analysis (EDA) on a residential real estate dataset to understand the key factors influencing property pricing. By cleaning and visualizing the data, we aim to uncover trends related to location, property size, and amenities.

## 📌 Problem Statement
To analyze how flat prices vary based on area, location (zone/city), number of bathrooms, balcony availability, and price per square foot, and to identify the major factors influencing property pricing.

## 🎯 Objectives
- Perform data cleaning and preprocessing for analysis.
- Identify key factors affecting property prices (Area, Location, Amenities).
- Visualize trends and correlations within the real estate market.
- Provide data-driven business recommendations for investors.

## 📊 Dataset Information
The analysis is based on a dataset containing 14,528 observations with the following features:
- Name: Name of the residential project.
- Property Title: Description of the flat/house type.
- Price: Property price (converted to Lakhs for analysis).
- Location: Specific area/city.
- Total_Area: Total area in square feet.
- Price_per_SQFT: Cost per square foot.
- Baths: Number of bathrooms.
- Balcony: Presence of a balcony (Yes/No).

## 🛠️ Tech Stack
- Language: Python
- Libraries: Pandas, NumPy, Matplotlib, Seaborn

## 🔍 Key Insights from EDA
- Property Size: There is a strong positive correlation between total area and price.
- Amenities: Properties with more bathrooms and balconies tend to command higher market prices.
- Location: Price per square foot varies significantly across different zones and locations.
- Value Drivers: Larger properties generally generate higher total value in the real estate market.

## 💡 Business Recommendations
- Investors should focus on locations with higher price per square foot growth.
- Larger properties tend to generate higher total value in the real estate market.
- Properties with additional amenities such as balconies and bathrooms show higher price trends.
  
📝 Conclusion
- This analysis highlights how property size, location, and amenities influence real estate pricing.
- The project demonstrates how data cleaning, exploratory data analysis, and visualization techniques can be used to extract valuable insights from real estate data.
